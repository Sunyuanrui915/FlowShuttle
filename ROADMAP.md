# Roadmap

## v0.1.0

目标：完成 Flow Shuttle 的首个公开版本。

计划内容：

* 完成项目命名统一：流梭 / Flow Shuttle / flowshuttle
* 完成 GitHub 私有仓库初始化
* 补齐 AGPL-3.0-only 许可证
* 补齐 README、CONTRIBUTING、CHANGELOG、ROADMAP、AGENTS.md
* 完成开源前安全检查，排除本地数据、密钥、日志、构建产物等不应提交内容
* 整理基础发布说明
* 准备 Windows 安装包
* 发布首个 GitHub Release：`v0.1.0`
* 仓库公开后，收集用户反馈和 Issue

## 后续版本方向

* 优化安装和升级体验
* 完善截图、使用说明和故障排查文档
* 根据用户反馈优化 Today、日报/周报/月报、热力图和项目时间线
* 评估是否需要多平台安装包
* 评估是否需要更完整的自动更新机制
