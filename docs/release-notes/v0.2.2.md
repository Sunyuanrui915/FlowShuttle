# Flow Shuttle v0.2.2 Release Notes

流梭 Flow Shuttle v0.2.2 是一个小版本修复与体验优化更新，主要修复编辑页空行保存丢失问题，并优化版本更新说明展示和作者联系方式入口。

## 修复

- 修复编辑页中输入空行后，保存并重新进入时空行丢失的问题。

## 优化

- Settings 页面“版本与更新”卡片现在只展示一句版本摘要，完整说明可通过 Release 详情查看。
- 新增“作者与反馈 / 关于流梭”入口，可查看个人网站、流梭页面、公众号和小红书信息。
- 移除 Settings 外观显示模式下方“当前选择 / 当前使用”的状态提示文字。

---

## English

Flow Shuttle v0.2.2 is a small bug-fix and experience update focused on preserving blank lines in the editor, simplifying release note display in Settings, and adding an author and feedback entry.

### Fixed

- Fixed an issue where blank lines entered in the editor could disappear after saving and reopening the entry.

### Improved

- The Settings update card now shows only a one-sentence release summary, with full notes available from the Release details page.
- Added an Author & Feedback entry with the personal website, Flow Shuttle page, WeChat Official Account, and Xiaohongshu information.
- Removed the extra current theme status text under the appearance selector in Settings.
