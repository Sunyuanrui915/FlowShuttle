# Flow Shuttle v0.1.1

流梭 Flow Shuttle 的 v0.1.1 版本，主要包含编辑器体验优化、UI 细节优化和若干发布前后的体验修复。

## Highlights

- 升级今日记录与备忘录的 Markdown 编辑体验，支持更稳定的标题、列表、任务列表、引用、代码块和高亮块编辑。
- 补充编辑器右键菜单、剪贴板文本操作、图片粘贴、图片复制和另存能力。
- 优化今日记录编辑页顶部信息区，减少空白并改善保存状态与操作区布局。
- 修正热力图字数统计口径，将工作项当前内容纳入统计，使当日记录字数更接近实际输入内容。
- 更新 v0.1.1 Windows 安装包与发布说明。

## Notes

- 数据默认保存在本地。
- 不强制登录，不上传工作内容。
- 当前 Windows 安装包暂未进行代码签名，首次安装可能出现 Windows 安全提示。
- 自动更新框架已预留，后续版本将继续验证更新体验。
