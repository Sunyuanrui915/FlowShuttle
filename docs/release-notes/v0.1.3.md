# Flow Shuttle v0.1.3 Release Notes

## 中文

流梭 Flow Shuttle v0.1.3 是一个小版本修复更新，主要修复版本更新说明显示、上一工作日参考内容展示，以及保存本项后 Today 状态刷新问题。

### 修复

- 修复 Settings 中 Release 说明直接显示 HTML 标签的问题。
- 修复上一工作日参考 / 上一快照中 Markdown 换行和 attachment 图片显示异常的问题。
- 修复今日记录编辑页点击「保存本项」后，Today 总览仍显示摘要未填的问题。

### 安装包

Windows 安装包：

`FlowShuttle-0.1.3-setup.exe`

SHA256：

`14e3642050f7a0e3c1298a95870458abe70a10748125da8d7b0f6f36da7b410d`

请确认下载来源为 GitHub Releases 官方页面：

https://github.com/Sunyuanrui915/FlowShuttle/releases

### 说明

- 本次更新不会修改本地数据目录。
- 建议升级前保留数据目录备份。
- 数据默认保存在本地。
- 不强制登录，不上传工作内容。
- 当前 Windows 安装包暂未进行代码签名，首次安装可能出现 Windows 安全提示。

## English

Flow Shuttle v0.1.3 is a small bug-fix release focused on release note rendering, previous workday reference rendering, and Today overview refresh after saving an entry.

### Fixed

- Fixed release notes being displayed as raw HTML in Settings.
- Fixed Markdown line breaks and attachment images not rendering correctly in previous workday references / previous snapshots.
- Fixed Today overview not recognizing saved change summaries after clicking Save Item.

### Installation

Windows installer:

`FlowShuttle-0.1.3-setup.exe`

SHA256:

`14e3642050f7a0e3c1298a95870458abe70a10748125da8d7b0f6f36da7b410d`

Please download from the official GitHub Releases page:

https://github.com/Sunyuanrui915/FlowShuttle/releases

### Notes

- This update does not change the local data directory.
- Keeping a backup of your data directory before upgrading is still recommended.
- Data is stored locally by default.
- Login is not required, and Flow Shuttle does not upload your work content by itself.
- The current Windows installer is not code-signed yet, so Windows may show a security warning on first install.
