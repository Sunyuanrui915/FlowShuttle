# Flow Shuttle v0.1.0

流梭 Flow Shuttle 的第一个公开版本。

## Highlights

- 本地优先的个人工作进展记录
- 项目与工作项管理
- Today 每日记录
- 日报 / 周报 / 月报生成
- 项目时间线
- 工作活跃度热力图
- Markdown 导出
- 本地 SQLite 存储
- Windows 安装包
- 预留自动更新框架

## Installation

普通用户请下载 Windows 安装包：

```text
FlowShuttle-0.1.0-setup.exe
```

SHA256：

```text
b78d35921eb6e37a32c31833972006aecc5c04e7f15c71fbbabec3973d2aba09
```

请确认下载来源为 GitHub Releases 官方页面：

```text
https://github.com/Sunyuanrui915/FlowShuttle/releases
```

## Notes

- 当前版本为 v0.1.0，是流梭的首个公开版本，仍处于早期阶段。
- 数据默认保存在本地。
- 不强制登录，不上传工作内容。
- 当前 Windows 安装包暂未进行代码签名，首次安装可能出现 Windows 安全提示。
- 自动更新框架已预留，真实自动更新将在后续公开版本中继续验证。
- 建议先使用非敏感数据体验。
