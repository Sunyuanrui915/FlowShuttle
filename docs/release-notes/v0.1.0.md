# Flow Shuttle v0.1.0

流梭 Flow Shuttle 的第一个公开版本。

## Highlights

- 本地优先的个人工作进展记录
- 项目与工作项管理
- Today 每日记录
- 日报 / 周报 / 月报生成
- 项目时间线
- 工作活跃度热力图
- Markdown 导出
- 本地 SQLite 存储
- Windows 安装包
- 预留自动更新框架

## Installation

普通用户请下载 Windows 安装包：

```text
FlowShuttle-0.1.0-setup.exe
```

## Notes

- 当前版本为 v0.1.0，仍处于早期公开阶段。
- 数据默认保存在本地。
- 不强制登录，不上传工作内容。
- 未进行代码签名，Windows 可能出现安全提示。
- 建议先使用虚拟数据或非敏感数据体验。
