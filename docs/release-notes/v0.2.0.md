# Flow Shuttle v0.2.0 Release Notes

## 中文

流梭 Flow Shuttle v0.2.0 是一次基础体验优化版本，重点修复热力图统计异常，并优化编辑器、搜索、上一工作日参考、Today 返回体验和项目 / 工作项管理方式。

### 修复

* 修复只点击“保存本项”、但没有填写今日变更摘要时，总记录字数和热力图活跃值异常升高的问题。
* 修复上一工作日参考中换行和分段显示不一致的问题，减少次日回看时的阅读干扰。

### 优化

* 优化上一工作日参考展示方式，不再单独标记“换行 / 分段”，只保留日期、上一工作日当前内容、变更摘要、下一步计划、阻碍 / 需要帮助五个部分。
* 取消编辑器拼写检查，减少中文、项目名、技术词汇和变量名下的红线干扰。
* 搜索结果增加关键词高亮显示，帮助更快定位命中内容。
* 搜索框增加清空按钮，便于快速重置搜索状态。
* 限制应用单实例运行，避免同时打开多个流梭窗口导致本地数据写入冲突。
* 简化编辑器换行逻辑，回车即自然换行，并移除“换行 / 分段”操作提示。
* 从今日记录编辑页返回 Today 时，保持离开前的滚动位置，不再自动跳转到刚编辑的工作项。
* 增加项目和工作项的手动顺序调整能力，常用项目和重点工作项可以更灵活地排在前面。

## English

Flow Shuttle v0.2.0 is a foundational experience update focused on heatmap statistics, editor behavior, search, previous workday references, Today navigation, and project / work item ordering.

### Fixed

* Fixed an issue where clicking “Save Item” without a valid change summary could incorrectly increase total text length and heatmap activity score.
* Fixed inconsistent line break and paragraph rendering in previous workday references, making past records easier to review.

### Improved

* Simplified the previous workday reference view by removing separate line-break / paragraph markers and keeping only date, previous content, change summary, next steps, and blockers.
* Disabled spell checking in editor fields to reduce noise for Chinese text, project names, technical terms, and variable names.
* Added keyword highlighting in search results for easier matching and navigation.
* Added a clear button to the search box.
* Prevented multiple Flow Shuttle instances from running at the same time to avoid local data write conflicts.
* Simplified editor behavior so Enter creates a natural line break, and removed line-break / paragraph operation hints.
* Preserved the Today page scroll position when returning from a work item editor, instead of jumping back to the edited item.
* Added manual ordering for projects and work items, making it easier to keep frequently used projects and important work items near the top.
