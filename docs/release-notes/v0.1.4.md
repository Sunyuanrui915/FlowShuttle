# Flow Shuttle v0.1.4 Release Notes

## 中文

流梭 Flow Shuttle v0.1.4 是一个体验优化版本，主要修复 Today 状态识别、上一工作日参考换行显示等问题，并补充工作项编辑能力。

### 修复

- 修复没有今日变更摘要记录时，Today 页面仍显示“摘要已填”的问题。
- 修复换行保存后，第二天在上一工作日参考中显示成分段的问题。

### 优化

- 增加换行与分段操作区分提示，帮助用户理解编辑器中的换行行为。
- 优化图片右键菜单，选中图片时不再显示“粘贴”“粘贴为纯文本”等不适合图片场景的操作。
- 增加工作项编辑能力，支持修改工作项标题、描述和状态，误标记为已完成后也可以改回。
- 优化部分 UI 显示细节，让日常记录和查看体验更顺畅。

### 升级说明

- 本次更新不会修改你的本地数据目录。
- 升级前仍建议保留数据目录备份。

## English

Flow Shuttle v0.1.4 is an experience-focused update that fixes Today state detection, previous workday reference rendering, and adds work item editing support.

### Fixed

- Fixed an issue where the Today page could show a change summary as filled even when no valid summary existed.
- Fixed an issue where line breaks could be rendered as separate paragraphs in the previous workday reference.

### Improved

- Added guidance to clarify line breaks and paragraph breaks in the editor.
- Improved the image context menu by removing paste actions that do not apply to selected images.
- Added work item editing, including title, description, and status updates, so completed items can be changed back when needed.
- Improved several UI display details for a smoother daily recording and review experience.

### Upgrade Notes

- This update does not change your local data directory.
- Keeping a backup of your data directory before upgrading is still recommended.
